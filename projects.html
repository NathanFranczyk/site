Projects Showcase: The highlights

Fuse Analysis

One of my tasks at Ameren was to, by hand, check all 1500+ substations' high side fuses to ensure that the fuse protects the transformer. I, instead, decided to use VBA in excel to write a program to do power calculations and select the best possible fuse to use. I had to see if the current fuse would protect the transformer, pull data from a few databases to calculate infinite bus fault current, and consult the SandC company for fuse data. Then, selecting the best fuse was easy.

PCR Machine

The Weizmann Institue asked me to design a device that expands their group's prototype's capabilities; task was to expand this prototype to be able to accomodate 8 samples at once and make the system portable. To make the system portable, I must produce a discrete circuit capable of automating the PCR, a power circuit to provide energy, and an optical system that can measure the DNA in a confined space.

Photomosaic

In my data structures class (using C++), we were tasked with designing a photomosaic that would read an input picture, decompose it into sections of a K-D tree , find pictures that most closely resemble the section, and replace the sections with the pictures.

2048

With C, I recreated the logic behind the popular game of 2048. This was part of a class project, and front end display was already created for me. This version allows a variable grid size. This game has been programmed with a dynamically allocated memory game board. This game board is constantly modified during the game's moving functions.

Maze Creator and Solver

Another project in my data structers class was to make a program that randomly creates and solves a maze based on a seed. This project used a disjoint sets data structure to see if sections of a maze were connected in order to create the maze. To solve the maze, a breadth first search was used to find the solution to the maze.

Audio Denoising

In my digital signal processing lab class, we were given an audio clip with added noise. My task was to graph this signal (in the frequency domain) and eliminate the noise using Matlab. I graphed the clip using a spectrogram and filtered out the high pitched noise.

Flower Power Energy

A few years ago I created a start-up that makes portable solar power generator retrofits. Because solar power generators already exist, I decided that the ability to hook up to a portable gas generator would help us differentiate myself from current portable solar units.

AM Radio

As a lab portion for my analog circuits class, one of our projects was to create a portion of an AM Radio. With an antenna, I fed the signal into a band passs filter to attenuate non radio frequencies and then amplifying the signal in an effort to reduce noise. Using a function generator as a local oscilator, I passed the signal into a prebuilt mixer circuit. After filtering non-audible noise, this signal was amplified and projected to headphones.
